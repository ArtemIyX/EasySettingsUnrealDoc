Detailed Description

A library class providing utility functions for the Easy Settings plugin.

This class provides static methods for accessing settings-related classes and developer settings in the Easy Settings plugin.

static FString UEasySettingsLib::GetConfigPath ( )
static

Retrieves the configuration path based on whether the editor is active or not.

This function returns the project configuration path if the editor is active (i.e., IsWithEditor() returns true). Otherwise, it returns the project directory path.
