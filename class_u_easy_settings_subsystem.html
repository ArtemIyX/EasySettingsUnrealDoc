Detailed Description

UEasySettingsSubsystem

A subsystem designed to handle various graphical and system settings within the game. It allows the user to get and set settings related to graphics quality, resolution, VSync, frame rate limits, and window modes. This subsystem is intended to be used within a game instance to control and apply user settings.
